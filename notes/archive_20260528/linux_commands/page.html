  • 1.新增內容進檔案:
    • echo '內容' > 檔案.py
  • 2.追加內容進檔案(最後一行):
    • echo '內容' >> 檔案.py
  • 3.$系列:
    • echo $USER
    • 確認你現在是誰（例如：kali）
    • echo $PATH
    • 看系統去哪裡找執行檔（資安必看，防止劫持）
    • echo $SHELL
    • 確認你目前用的是 zsh 還是 bash
  • 4.字串處理:
    • 換行顯示：echo -e "第一段\n第二段"
    • 關鍵：-e 是啟動開關，讓系統看得懂 \n（換行）或 \t（縮排）
    • 密碼處理：echo -n "admin"
    • 關鍵：-n 會去掉結尾的換行符號。這在計算 MD5 或 Base64 雜湊值時非常重要，多一個換行符號結果就會完全不同
  • 5.pipe  |

              密碼/字串編碼：
 
    • echo -n "text" | base64 → 快速產生編碼
    • echo "base64_str" | base64 -d → 快速還原
    • 安全雜湊計算：
    • echo -n "pass" | md5sum → 獲取 MD5 值
    • 字串修改 (sed)：
    • echo "old text" | sed 's/old/new/' → 快速替換文字
    • 自動化傳輸：
    • echo "command" | python3 → 不存檔直接跑 Python 代碼

閱讀和處理與串聯資料流：
  • cat (Concatenate)
  • 1. 真正的核心功能：串聯檔案 (Concatenate)
  • 這才是它名字的由來。如果你有三個碎掉的 Base64 檔案，你可以用 cat 把它們接成一個完整的檔案：
    • 指令：cat part1.txt part2.txt part3.txt > full_secret.txt
    • 用途：將多個小檔案合併成一個大檔案，這在處理被分割過的惡意程式或日誌檔（Log files）時非常有用。
  •  2. 快速建立新檔案
  • 不需要打開文字編輯器（如 Mousepad 或 Leafpad），你可以直接用 cat 寫入文字：
    • 指令：cat > note.txt
    • 動作：輸入指令後，螢幕會變空白，這時你可以開始打字。打完後按下 Ctrl + D 儲存。
    • 用途：在滲透測試中，當你拿到一個只有終端機的權限時，這是寫入 Payload 最快的方法。
  • 3. 把內容「倒」進另一個指令（資料流處理）
  • 這就是我們之前聊到的 Pipe (|) 的好夥伴：
    • 指令：cat secret.b64 | base64 -d
    • 原理：cat 負責把二進位或文字流「噴」出來，再交給下一個工具處理。雖然直接用 base64 -d secret.b64 也可以，但用 cat 的寫法能讓你更容易串接多層過濾器。
  •  4. 顯示隱形的「危險」符號
  • 這是資安人員最愛的隱藏功能。有些惡意代碼會藏在大量的空格或換行符號裡，或者利用特殊字元讓 cat 顯示出來的結果騙過你的眼睛。
    • 指令：cat -A filename
    • 效果：
    • 換行符號會顯示為 $。
    • Tab 鍵會顯示為 ^I。
    • 用途：辨識程式碼注入。如果你發現檔案結尾出現一堆奇怪的 ^M 或空格，通常代表這份檔案被動過手腳。
    •  
  •  
  • 常用參數：
  • -n：顯示行號（找程式碼漏洞時好用）。
  • -A：顯示所有隱形字元（包含換行、Tab）。
  • -s：壓縮連續的空白行，讓畫面變整潔。
  • 經典組合：
  • cat [檔案] | grep [關鍵字]：在檔案裡搜尋
  • cat << EOF > file.txt：在腳本中自動化產生檔案（進階用法）。

指令組合技：Pipe vs Chain
  • | (Pipe)：資料傳遞。將 A 的輸出內容丟給 B 處理。
    • 範例：cat file.txt | grep "secret"
  • && (AND)：邏輯執行。A 成功執行完成後，才執行 B。
    • 範例：sudo updatedb && locate my_payload
  • || (OR)：失敗備援。A 執行失敗了，才執行 B。

 指令：adduser
  • 功能：以互動方式新增系統使用者。
  • 常用流程：
    • sudo adduser [username]：新增帳號。
    • sudo deluser --remove-home [username]：刪除帳號並移除家目錄。
  • 相關檔案：
    • /etc/passwd：使用者基本資訊。
    • /etc/shadow：加密後的密碼（只有 root 可讀）。
    • /etc/group：群組資訊。
grep指令用法說明
  • grep "關鍵字" 檔案在指定檔案中搜尋關鍵字。
  • grep -i "abc" 檔案忽略大小寫（Ignore case），找 ABC 或 abc 都行。
  • grep -r "密碼" /etc/遞迴搜尋（Recursive），搜尋 /etc/ 底下所有子資料夾的檔案。
  • grep -n "root" 檔案顯示符合內容的行號（Line number）。
  • grep -c "kali" 檔案計算出現次數（Count），不看內容只看次數。
