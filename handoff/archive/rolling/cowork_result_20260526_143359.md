**1. Right next step?** Yes. Worker attestation validates one artifact but is role-agnostic. The synthesizer in `scripts/live-bounty-preview-synthesize.py` already expects multi-reviewer JSON companion packets, yet no schema defines what a single role's packet must contain, and no manifest ties multiple role packets into one Hermes decision. This slice closes that gap with the same `bin/hermes review` enforcement pattern that just landed for attestation.

**2.
