# Public Filtered Obsidian Memory Export

This folder contains sanitized project-memory notes exported from the external Obsidian vault.

Policy:

- note/content files are kept where possible;
- `.obsidian` workspace/plugin/config files are excluded;
- private paths, target/program names, local IPs, aliases, secrets, and specific advisory IDs are redacted;
- raw evidence, credentials, browser state, and app-local settings are not included.

Skipped app/config/non-note files: 10
